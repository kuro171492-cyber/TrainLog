# TrainLog Server Launcher

Простой способ запустить TrainLog на телефоне через локальную сеть.

## Быстрый старт

**Просто двойной клик по файлу:**
```
start_server.bat
```

Откроется окно с управлением сервером.

## Возможности интерфейса

### Управление сервером
- **▶ Start Server** — запустить веб-сервер
- **⏹ Stop Server** — остановить сервер
- **Status** — текущее состояние

### Доступ с телефона
Приложение автоматически определяет IP-адрес вашего компьютера и отображает полный URL.

**Два способа отправить адрес на телефон:**

1. **📋 Copy URL** — скопировать в буфер обмена
   - Кликни кнопку
   - Адрес скопируется в буфер
   - Вставь его в браузер на телефоне

2. **📱 Generate QR Code** (опционально)
   - Сначала установи: `python -m pip install qrcode[pil]`
   - Кликни кнопку
   - Отсканируй QR-код с телефона

### Другие функции
- **Open in Browser** — открыть приложение на компьютере
- **Quit** — закрыть лаунчер

## Требования

- Python 3.6+ (с путём в PATH)
- Компьютер и телефон в одной WiFi-сети

## Установка QR-кода (опционально)

Если хочешь использовать кнопку "Generate QR Code", установи модуль:

```bash
python -m pip install qrcode[pil]
```

Без этого можно просто копировать URL и вставлять в браузер.

## Устранение неполадок

**Ошибка: "Python is not installed"**
- Установи Python с https://www.python.org/
- Убедись, что при установке отмечен "Add Python to PATH"

**Телефон не видит адрес**
- Убедись, что компьютер и телефон в одной WiFi-сети
- Не используй VPN или гостевые сети

**Server port already in use**
- Закрой другие приложения, использующие порт 8000
- Или измени порт в коде (переменная `self.port`)
